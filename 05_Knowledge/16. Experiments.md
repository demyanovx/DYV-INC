## Исторический эксперимент №1 — Альфа Регбиз + РКО

| Показатель | Значение |
|---|---|
| Источник | Друзья / сарафанное радио |
| Канал | Личный кабинет банка |
| Ставка | 7 000 ₽ |
| Расход | 0 ₽ |
| Лиды | 4 |
| Оплаченные лиды | 4 |
| Апрув / выкуп | 100% |
| Подтверждённый доход | 28 000 ₽ |
| Доход на лида | 7 000 ₽ |
| ROI | Не рассчитывается при нулевом расходе |

**Вывод:** сарафанное радио подтвердило возможность привлекать оплачиваемых клиентов без рекламных расходов. Результат используется как исторический ориентир, но не как прогноз для платных источников.

## Эксперимент №2 — прямое привлечение лидов через Авито

| Параметр | Значение |
|---|---|
| Дата подготовки | 2026-07-12 |
| Ответственный | Павел |
| Статус | Платное продвижение запущено |
| Дата запуска | 2026-07-13 |
| Дата платного этапа | 2026-07-19 |
| Подтверждение источника | 2026-07-13 менеджер FCN подтвердил, что трафик с Авито разрешён без дополнительных ограничений |
| Источник | Авито, раздел услуг |
| Формат | Прямое объявление для конечного клиента |
| Основной запрос | Бесплатная помощь с регистрацией ИП и открытием расчётного счёта |
| География | Иркутск; сопровождение дистанционно |
| Органический этап | 4 просмотра, 0 избранных, 0 контактов, 0 заявок, расход 0 ₽ |
| Платный этап | 75 ₽ в день в течение 5 дней; общий лимит 375 ₽ |
| Прогноз Авито | 5–25 дополнительных просмотров |

### Гипотеза

Узкое объявление о регистрации ИП и подборе РКО даст квалифицированные прямые обращения без необходимости сначала искать рефероводов.

### Позиционирование

Помощь с регистрацией бизнеса и осознанным выбором РКО. Подбираются только подходящие клиенту продукты; гарантированное одобрение не обещается.

### Креативы

После органического этапа галерея полностью переработана в едином стиле:

1. «Регистрация ИП — 0 ₽»: дистанционно и с пошаговой помощью.
2. «Как всё проходит»: консультация → регистрация ИП → открытие счёта.
3. «Бесплатное сопровождение»: дистанционно, помощь на каждом этапе.

Визуальный стиль: формат 4:3, тёмно-синий и графитовый фон, бирюзовые акценты, крупная белая типографика и минималистичная fintech-графика. Не используются банковские логотипы, деньги, государственная символика и внутренние документы DYV INC.

### План запуска

1. 2026-07-13 получено подтверждение менеджера FCN об отсутствии ограничений на трафик с Авито.
2. Проведён органический этап без платного продвижения.
3. После 4 просмотров без контактов проблема локализована на уровне недостатка трафика.
4. Заголовок изменён на «Регистрация ИП бесплатно — открытие счёта».
5. Галерея заменена на три согласованные карточки формата 4:3.
6. 2026-07-19 запущено продвижение на 5 дней с лимитом 75 ₽ в день.
7. Во время платного этапа объявление не менять; снять промежуточную статистику через 48 часов и итоговую после завершения.

### Метрики

- просмотры объявления;
- добавления в избранное;
- входящие сообщения;
- конверсия просмотра в сообщение;
- квалифицированные лиды;
- стоимость квалифицированного лида;
- заявки Озон Регбиз + РКО;
- подтверждённый доход и ROI каскада.

### Результаты

#### Органический этап

- просмотры: 4;
- добавления в избранное: 0;
- контакты: 0;
- квалифицированные лиды: 0;
- заявки: 0;
- расход: 0 ₽.

Четырёх просмотров недостаточно для оценки конверсии предложения. Основная проблема первого этапа — недостаточная видимость объявления, а не подтверждённая слабость оффера.

#### Платный этап

Запущен 2026-07-19: 75 ₽ в день на 5 дней, общий лимит 375 ₽. Исходная точка перед продвижением — 4 просмотра и 0 контактов. Прогноз интерфейса Авито — 5–25 дополнительных просмотров.

#### Правила оценки

- при появлении целевого контакта рассчитать стоимость квалифицированного лида;
- при 15–25 дополнительных просмотрах без контактов пересмотреть предложение или первые строки описания;
- при 5–10 дополнительных просмотрах считать объём теста недостаточным и решать вопрос о втором цикле отдельно;
- не увеличивать бюджет автоматически и не считать заявку доходом до подтверждённой выплаты.
