## Дополнительные метрики каскада

Для экспериментов с банковскими офферами дополнительно фиксировать:

- Lead ID без персональных данных;
- источник и стоимость привлечения;
- количество оформленных продуктов на лида;
- количество подтверждённых продуктов;
- подтверждённый доход на лида;
- конверсию между этапами каскада;
- срок от привлечения до последней выплаты;
- причину остановки прохождения каскада.

## Исторический эксперимент №1 — Альфа Регбиз + РКО

| Показатель | Значение |
|---|---|
| Источник | Друзья / сарафанное радио |
| Канал | Личный кабинет банка |
| Ставка | 7 000 ₽ |
| Расход | 0 ₽ |
| Лиды | 4 |
| Оплаченные лиды | 4 |
| Апрув / выкуп | 100% |
| Подтверждённый доход | 28 000 ₽ |
| Доход на лида | 7 000 ₽ |
| ROI | Не рассчитывается при нулевом расходе |

**Вывод:** сарафанное радио подтвердило возможность привлекать оплачиваемых клиентов без рекламных расходов. Результат используется как исторический ориентир, но не как прогноз для платных источников.

| Параметр | Значение |

| География | Иркутск; сопровождение дистанционно |
| Бюджет первого этапа | 0 ₽ на первые 3 дня |
| Лимит платного теста | До 1 000 ₽ только после оценки органической выдачи |

### Гипотеза

Узкое объявление о регистрации ИП и подборе РКО даст квалифицированные прямые обращения без необходимости сначала искать рефероводов.

### Позиционирование

Помощь с регистрацией бизнеса и осознанным выбором РКО. Подбираются только подходящие клиенту продукты; гарантированное одобрение не обещается.

### Креативы

1. «Открываешь ИП? Не выбирай банк вслепую».
2. «Как работаем: разбор ситуации → регистрация бизнеса → подбор РКО».
3. «Формат работы: дистанционно → по шагам → без навязывания».

Визуальный стиль: чёрный, белый, графит, тёмно-синий и небольшой зелёный акцент; фотореализм; без банковских логотипов, официальной символики и крикливой рекламы.

### План запуска

1. Подтверждение получено 2026-07-13: менеджер FCN сообщил об отсутствии ограничений на трафик с Авито.
2. Опубликовать объявление без платного продвижения.
3. Проверить путь тестового обращения до создания лида и заявок в CRM 2.0.
4. Три дня фиксировать просмотры, добавления в избранное, сообщения и квалифицированные лиды.
5. Менять за один раз только один элемент: заголовок, первую карточку или категорию.
6. Платное продвижение до 1 000 ₽ тестировать только после органического этапа.

### Метрики

- просмотры объявления;
- добавления в избранное;
- входящие сообщения;
- конверсия просмотра в сообщение;
- квалифицированные лиды;
- стоимость квалифицированного лида;
- заявки Озон Регбиз + РКО;
- подтверждённый доход и ROI каскада.

### Результаты

Объявление опубликовано 2026-07-13 без платного продвижения. Первичная оценка органической выдачи — через 3 дня. До подтверждения выплаты заявки не считаются доходом.
