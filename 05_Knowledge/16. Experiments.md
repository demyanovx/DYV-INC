## Дополнительные метрики каскада

Для экспериментов с банковскими офферами дополнительно фиксировать:

- Lead ID без персональных данных;
- источник и стоимость привлечения;
- количество оформленных продуктов на лида;
- количество подтверждённых продуктов;
- подтверждённый доход на лида;
- конверсию между этапами каскада;
- срок от привлечения до последней выплаты;
- причину остановки прохождения каскада.

## Исторический эксперимент №1 — Альфа Регбиз + РКО

| Показатель | Значение |
|---|---|
| Источник | Друзья / сарафанное радио |
| Канал | Личный кабинет банка |
| Ставка | 7 000 ₽ |
| Расход | 0 ₽ |
| Лиды | 4 |
| Оплаченные лиды | 4 |
| Апрув / выкуп | 100% |
| Подтверждённый доход | 28 000 ₽ |
| Доход на лида | 7 000 ₽ |
| ROI | Не рассчитывается при нулевом расходе |

**Вывод:** сарафанное радио подтвердило возможность привлекать оплачиваемых клиентов без рекламных расходов. Результат используется как исторический ориентир, но не как прогноз для платных источников.
