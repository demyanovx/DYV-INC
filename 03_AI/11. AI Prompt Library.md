# DYV INC — AI Prompt Library



## Назначение



Этот документ содержит лучшие промпты, шаблоны и рекомендации по использованию AI-инструментов в проекте DYV INC.



Главная цель — использовать каждый инструмент только для тех задач, в которых он показывает лучший результат.



---



# ChatGPT



## Основные задачи



- написание постов;

- сценарии YouTube;

- сценарии Shorts;

- контент-планы;

- маркетинг;

- анализ статистики;

- анализ конкурентов;

- создание стратегий;

- упаковка Telegram;

- генерация идей;

- исследования;

- написание SOP;

- написание инструкций;

- помощь в переговорах;

- анализ воронок;

- генерация промптов.



## Когда использовать



Всегда, когда требуется глубокий анализ, структурирование информации или создание текстового контента.

# Общие правила



Перед выбором инструмента ответить:



1. Что нужно получить?



Текст?



Изображение?



Видео?



Исследование?



2. Какой инструмент делает это лучше остальных?



Использовать лучший инструмент, а не привычный.
